## Segment Type

`text`

## What

Display text.

## Sample Configuration

```yaml
prompt:
  - segments: ["text"]

text:
  type: "text"
  style: "plain"
  foreground: "#E06C75"
  template: "❯"
```

## Template

### Properties

Text segments have no special options. See [Templates](../../configuration/templates.md) for globally available options.
