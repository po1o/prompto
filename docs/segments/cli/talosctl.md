## Segment Type

`talosctl`

## What

Displays the currently active [Talosctl][talosctl] context name.

This segment leverages the `.talos/config` file generated by the [Talosctl CLI][talosctl]. The CLI docs can be found by
going to Documentation > Reference > CLI. For example, [v1.7 CLI][CLI].

## Sample Configuration

```yaml
prompt:
  - segments: ["talosctl"]

talosctl:
  type: "talosctl"
  style: "powerline"
  powerline_symbol: ""
  foreground: "#FFFFFF"
  background: "#FF6C00"
  template: " 󱃾 {{ .Context }}"
```

## Template

### Default Template

```template
 {{ .Context }}
```

### Properties

- `.Context`
  - Type: `string`
  - Description: the current talosctl context
