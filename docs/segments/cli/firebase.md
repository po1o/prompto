# Firebase

## Segment Type

`firebase`

## What

Display the current active [Firebase][firebase] project.

This segment leverages the `firebase-tools.json` file generated by the [Firebase CLI][firebase-cli-docs]. The active
files listed there are used to determine the current project against the current directory.

## Sample Configuration

```yaml
prompt:
  - segments: ["firebase"]

firebase:
  type: "firebase"
  style: "powerline"
  powerline_symbol: ""
  foreground: "#ffffff"
  background: "#FFA000"
  template: " 󰥧 {{ .Project }}"
```

## Template

### Default Template

```template
{{ .Project }}
```

### Properties

- `.Project`
  - Type: `string`
  - Description: the currently active project

[firebase]: https://firebase.google.com/
[firebase-cli-docs]: https://firebase.google.com/docs/cli
